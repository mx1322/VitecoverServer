# VitecoverServer

This repository is now organized as a small monorepo:

- `backend/`
  Directus-based insurance backend, local Docker stack, schema sync script, and deployment-oriented backend assets.
- `frontend/`
  Reserved for the website and customer/admin-facing frontend application.
- `docs/`
  Product, schema, and local workflow documentation.

## Current focus

The backend is already initialized around the temporary vehicle insurance workflow:

- customer accounts
- vehicles
- drivers
- insurance products
- geo zones
- pricing rules
- quotes
- orders
- payments
- admin reviews
- policies
- refunds
